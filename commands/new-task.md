# /new-task

Create a new task file from the template.

Required information:

- Task ID (next sequential number)
- Goal (one sentence)
- Scope (bullet list of what's included)
- Acceptance criteria (executable — what checks/conditions prove "done")
- Worker role required
- Stop conditions
- Contracts referenced

Output path: `tasks/TASK-XXX-<slug>.md`

Use template from `templates/task.md`.
